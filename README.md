# 🎮 Guardiões da Floresta

Um jogo simples desenvolvido em Python com Pygame.

## 📋 Sobre o Jogo
Você controla um esquilo que deve desviar das raposas! Cada colisão com uma raposa custa uma vida. Você tem 5 vidas. Boa sorte!

## 🎯 Controles
- **W** - Mover para cima
- **S** - Mover para baixo  
- **A** - Mover para esquerda
- **D** - Mover para direita
- **ENTER** - Iniciar jogo
- **ESC** - Sair

## 🚀 Como Executar

### Pré-requisitos
- Python 3.11 ou superior
- Pip (gerenciador de pacotes do Python)

### Instalação
```bash
# 1. Clone o repositório
git clone https://github.com/SEU_USUARIO/guardiões-da-floresta.git

# 2. Entre na pasta do projeto
cd guardiões-da-floresta

# 3. Instale as dependências
pip install pygame

# 4. Execute o jogo
python main.py
